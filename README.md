# Tic-Tac-Toe-game
A simple Tic Tac Toe Game built using HTML, CSS, and JavaScript where two players take turns marking X and O on a 3×3 grid. The game automatically detects the winner or a draw and provides an interactive user interface.

# ❌⭕ Tic Tac Toe Game

A classic **Tic Tac Toe** web game where two players compete by placing **X** and **O** on a 3×3 grid. The goal is to align three marks in a row, column, or diagonal to win the game.

This project demonstrates **JavaScript game logic, DOM manipulation, and event handling**.

---

## 🚀 Features

* Two-player gameplay
* Automatic winner detection
* Draw detection when grid is full
* Restart / Reset game option
* Clean and responsive interface
* Beginner-friendly JavaScript project

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the game
* **CSS3** – Styling and grid layout
* **JavaScript** – Game logic and interactions

---


## 📌 Game Rules

* The game is played on a **3×3 grid**
* One player uses **X** and the other uses **O**
* Players take turns placing their marks
* The first player to get **three marks in a row** (horizontal, vertical, or diagonal) wins
* If all boxes are filled without a winner, the game ends in a **draw**
